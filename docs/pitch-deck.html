03 · A

Solução — O Produto

O que é o TGA

TGA é um protocolo de infraestrutura onchain que transforma o mercado brasileiro de garantias locatícias em um ativo de rendimento real acessível globalmente.

Opera em duas frentes simultâneas: B2B — integra imobiliárias com infraestrutura de registro e liquidação automática; B2C — emite o token TGA com NAV apreciável para investidores.

Estrutura do fundo — TGA SA (Segregated Account)
80%
Camada Ativa

Alocada em contratos de garantia registrados via SGR. Cada contrato gera receita que retorna ao fundo e eleva o NAV.

20%
Camada de Liquidez

Mantida em stablecoins gerando yield passivo. Buffer para resgates sem comprometer a operação ativa.

NAV Apreciável

O retorno não é distribuído — é incorporado ao fundo. Quanto mais contratos ativos, maior o NAV. O investidor captura o crescimento via valorização do token.

03 · C

Solução — Infraestrutura

SGR
Sistema de Garantia Registrada

Cada contrato de garantia é registrado onchain via smart contracts. A imobiliária acessa via dashboard e registra o contrato — que passa a existir como um registro imutável na blockchain, com hash criptográfico verificável.

✓ Lastro do fundo verificável em tempo real
✓ Sem exposição de dados sensíveis
✓ Auditável publicamente por qualquer participante
CLP
Camada de Liquidação Programável

Quando há inadimplência, uma integração offchain aciona o smart contract responsável, que valida as condições e executa a liquidação automaticamente. O repasse vai direto à imobiliária para repasse ao proprietário.

Go to Market

FASE 1 — PILOTO
Imobiliárias — entrada via rede existente
