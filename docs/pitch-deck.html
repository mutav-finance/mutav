03 · A

Solution — The Product

What is TGA

TGA is an onchain infrastructure protocol that turns the Brazilian rental guarantee market into a real-yield asset accessible globally.

It operates on two simultaneous fronts: B2B — integrates real estate agencies with automated registration and settlement infrastructure; B2C — issues the TGA token with an appreciating NAV for investors.

Positioning
The only onchain protocol with direct exposure to the Brazilian rental guarantee market.

Doesn't compete with real estate agencies — operates as the infrastructure layer connecting them to global capital.

Analogy

Think of TGA as OnRe — tokenized reinsurance — but the underlying asset is the Brazilian rental guarantee market, with onchain-verifiable collateral via SGR.

Fund structure — TGA SA (Segregated Account)
80%
Active Layer

Allocated to guarantee contracts registered via SGR. Each contract generates revenue that returns to the fund and raises the NAV.

20%
Liquidity Layer

Held in stablecoins generating passive yield. Buffer for redemptions without compromising active operations.

Appreciating NAV

Returns aren't distributed — they're built into the fund. More active contracts means a higher NAV. Investors capture growth through token appreciation.

03 · C

Solution — Infrastructure

SGR
Registered Guarantee System

Each guarantee contract is registered onchain via smart contracts. The agency accesses it via dashboard and registers the contract — which then exists as an immutable record on the blockchain, with a verifiable cryptographic hash.

✓ Fund collateral verifiable in real time
✓ No sensitive data exposure
✓ Publicly auditable by any participant
CLP
Programmable Settlement Layer

When there's a default, an offchain integration triggers the responsible smart contract, which validates the conditions and automatically executes settlement. The payment goes directly to the agency for forwarding to the landlord.
